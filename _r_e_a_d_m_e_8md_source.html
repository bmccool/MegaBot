5 ### What is this repository for? ###
6 
7 This repository is for the BeagleVision project. The goals of this project are to create a system that is capable of playing simple video games based on input from a camera used to discern the state of the game. For more info, see the [notebook](https://onedrive.live.com/redir?resid=39227574E31A4574!675&authkey=!AOqkbbqrZhCvqdE&ithint=onenote%2c).

10 ### How do I get set up? ###
11 
12 The setup requires a BeagleBoneBlack Rev C, a compatible webcam (Playstation Eye), and an internet connection. To get started, simple clone the project to the BeagleBone, run make, and run the executable BeagleVision. The BeagleBone is currently using the pre-installed Debian OS, which includes the OpenCV libraries.
13 
14 edit: To facilitate debugging and faster development, the project is currently being used on an Ubuntu Desktop installation. This is done so that we have more robust display capabilities and faster compile time.
